About This Application

Welcome to the Personal To-Do List App! This is a simple task management application built with Flask as part of the Part 6 homework exercise.

Features

  • View all your tasks in one place
  • Filter tasks by priority (High, Medium, Low)
  • View detailed information about each task
  • Add new tasks with a simple form
  • Color-coded status and priority indicators

Technologies Used

  • Flask - Python web framework
  • Jinja2 - Template engine for dynamic HTML
  • HTML/CSS - Frontend structure and styling
