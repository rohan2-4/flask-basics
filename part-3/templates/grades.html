How It Works

In the Flask app, we pass a list of dictionaries:

return render_template('grades.html',
    grades=[
        {'subject': 'Math', 'grade': 'A'},
        {'subject': 'Science', 'grade': 'B+'},
    ])

Jinja2 For Loop

We use a Jinja2 for loop to display each grade:

{% for grade in grades %}
<tr>
<td>{{ grade.subject }}</td>
<td>{{ grade.grade }}</td>
</tr>
{% endfor %}
